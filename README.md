# fdellabetta/tcpserver
Alpine-based image with just tcpserver by D. J. Bernstein (https://cr.yp.to/ucspi-tcp/tcpserver.html)

## Usage

```console
$ docker run -it -p 5678:5678 --rm dellabetta/tcpserver 0 5678 echo 5678
```
